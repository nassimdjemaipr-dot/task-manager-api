# Task Manager API

A simple task manager REST API built with Express.js.

## Install

```bash
npm install
```

## Run

```bash
npm start
```

## Endpoints

- `POST /login` - Login
- `GET /profile/:username` - User profile
- `GET /tasks/search?q=` - Search tasks
- `POST /tasks` - Create task
- `DELETE /tasks/:id` - Delete task
- `GET /admin/users` - List users
- `GET /ping?host=` - Ping host
- `GET /files?name=` - Read file
